Tech Stack (recommended, swappable)

  • CMS / Course host: Kajabi (preferred - all-in-one) or Teachable
  • Landing pages: Webflow (control) or Kajabi (speed)
  • Email: Kit / ConvertKit (preferred for creator audiences) or Kajabi-native
  • Payment: Stripe (course) + Whop (community / membership)
  • Video host: Wistia (course) + YouTube (long-form, Frank-owned channel)
  • Analytics: GA4 + Meta Pixel + Microsoft Clarity (heatmaps)
  • CRM: GHL (preferred) or Kajabi-native
  • Community / High-tier: Skool (preferred - low overhead) or Circle
  • Scheduling: Calendly (default) or Cal.com
  • Project / build: Notion (handover-friendly)

Integration Points

  1. Checklist opt-in page triggers Kit on form submission with type tag (operator vs enthusiast)
  2. Email nurture triggers Kajabi enrolment on course purchase webhook
  3. Course completion routes to Best Pizza Community application form
  4. All events fire to GA4 + Meta Pixel via GTM for full funnel attribution
  5. Vice content clips link to checklist opt-in in descriptions and text overlays
  6. Restaurant website embed triggers same Kit list with "restaurant-visitor" tag

Owner Map

  • Frank: filming (VSL + kitchen B-roll), monthly live Q&A on course, content appearance for IG, top-line decisions on copy + brand, Vice content licensing conversation
  • AG1 (Andrei): everything else - landing page build, email writing, automation setup, content production cycles, YouTube channel setup, Vice repurposing workflow, ads management, weekly reporting

Failure Modes + Mitigations

  • VSL filming delays → use existing Vice/Munchies episode clip as Phase 1 placeholder; replace with custom VSL by end of Phase 1
  • Vice licensing friction → start with new original short-form content first; add Vice clip repurposing once licensing is confirmed
  • Restaurant website update delays → run checklist opt-in as standalone page first, add restaurant embed later
  • Course content creation time → start with 2-module beta launch at $197, expand to full 6 modules over 60 days

KPIs (90-day)

  • Phase 1 · Best Pizza Operator Course landing live · 20+ buyers in first 30 days
  • Phase 2 · 500+ checklist opt-ins · 8-12% IG-to-email conversion on checklist
  • Phase 3 · 40+ short-form posts shipped · YouTube channel with 10+ videos live
  • Phase 4 · Meta CPL under $30 · ROAS 3x by week 12
  • Year-1 floor target · $240-600K incremental on top of existing restaurant revenue
